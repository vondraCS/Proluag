# Proluag

Proluag Library - Use Prolog in Luau

## Getting Started

### What is Proluag?

Proluag allows you to use Prolog through Luau, through a module script. The module script can be server or cilent sided, as long as you implement it properly.
It's a project I made to get better at both languages

### Installation

Simply download the Library.luau file, import into Roblox, and place it where you see fit (ReplicatedStorage, ServerStorage, etc.)

## Usage

### Types

Proluag introduces a few custom types, which are used in Prolog, including:

- Atoms
- Functors

As well as uses types that are already implemented in Lua, ie:

- Strings
- Floats
- Numbers
- etc.
